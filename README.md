## 什么是EFCore CLI

适用于Entity Framework Core的命令行接口(CLI)工具可执行设计时开发任务。例如，可以创建迁移、应用迁移，并为基于现有数据库的模型生成代码。

## 获取EFCore CLI

### 安装EFCore CLI

> 全局安装

```bash
dotnet tool install --global dotnet-ef
```
